This is an Android App that will be available on the Google Play Market. It will behave a lot like currently existing apps such as Siri and Google Now, with the exception that this app does not utilize internet of any kind - Wi-Fi nor 3G/4G/LTE.
The way this works is the app recieves the voice command and in the background sends a message to a designated number. This number is hosted by Twilio. When the Twilio server recieves the SMS, it forwards the message to my PHP code hosted on Heroku. My PHP code processes the information, sends a request back to the Twilio server, and the Twilio server then sends the text back to your Android phone in response to your query. The app then intercepts the message to display the information onto your phone as Siri or Google Now would. This operates a lot like Cosmos with the exception that this isn't a browser.
